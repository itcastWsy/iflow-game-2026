# 春节马年祝福海报生成器 2026

## 项目简介

春节马年祝福海报生成器是一个基于 HTML + CSS + Three.js 开发的春节主题互动网站。用户只需输入姓名，即可生成精美的马年祝福海报，并支持下载分享。

### 核心功能
- **个性化祝福**：根据用户姓名自动生成专属祝福语
- **30条祝福词库**：内置丰富马年祝福语，随机生成
- **3D炫酷背景**：Three.js 粒子系统 + 3D灯笼动画
- **多样海报风格**：8种渐变背景组合，每次生成都有惊喜
- **一键下载**：支持将海报下载为PNG图片
- **春节主题设计**：红灯笼、金色边框、马年主题元素

---

##  技术实现

### 技术栈
- **HTML5**：页面结构
- **CSS3**：动画效果、渐变、布局
- **Three.js (r128)**：3D背景动画
- **Canvas API**：海报绘制

### 核心功能实现

#### 1. Three.js 3D背景
- **粒子系统**：500个金色粒子在3D空间中旋转
- **3D灯笼**：5个红色灯笼悬浮在背景中，带有浮动动画
- **性能优化**：使用透明度和模糊效果，不影响前端性能

#### 2. 海报生成系统
```
海报生成流程：
1. 用户输入姓名 → 2. 随机选择祝福词 → 3. 随机选择渐变背景
→ 4. 绘制装饰元素 → 5. 绘制马年主题 → 6. 添加祝福词 → 7. 渲染完成
```

#### 3. 祝福词库设计
内置30条马年祝福语，采用模板替换机制：
```javascript
const blessings = [
    "{name}祝福马年大吉！",
    "{name}祝您马年马到成功！",
    // ... 更多祝福词
];
```

#### 4. Canvas绘制
- **渐变背景**：8种春节主题渐变色
- **装饰元素**：金色边框、角落花纹、红灯笼
- **文字渲染**：渐变色"马年"字样，带阴影效果
- **祝福词框**：半透明背景 + 金色边框

## 使用说明

### 运行方式
1. 下载项目文件到本地
2. 直接双击打开 `index.html` 文件
3. 或使用本地服务器启动：
   ```bash
   # 使用 Python 启动
   python -m http.server 8000
   
   # 使用 Node.js 启动（需安装 http-server）
   npx http-server
   ```

### 操作步骤
1. **输入姓名**：在输入框中填写姓名（支持中文，最多10字）
2. **生成海报**：点击"生成海报"按钮或按回车键
3. **预览效果**：查看生成的马年祝福海报
4. **重新生成**：点击"重新生成"可获取不同的祝福词和样式
5. **下载海报**：点击"下载海报"将图片保存到本地
6. **返回输入**：点击"返回输入"为其他人生成祝福

### 特色功能
- ✅ 支持回车键快速生成
- ✅ 空姓名输入验证
- ✅ 每次生成随机样式
- ✅ 图片文件名自动命名（"姓名_马年祝福海报.png"）
